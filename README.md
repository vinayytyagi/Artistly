# Artistly.com Frontend Assignment

A demo web app for a fictional performing artist booking platform, built with Next.js 13+, React, Tailwind CSS, and ShadCN UI.

## Assignment Overview

- 3-4 page functional, mobile-responsive demo
- Static JSON/mock API for data
- Focus: code structure, responsiveness, forms, filtering, and best practices

## Tech Stack
- Next.js 13+ (App Router)
- React Functional Components
- Tailwind CSS
- ShadCN UI
- React Hook Form + Yup

## Pages
- Homepage
- Artist Listing (with filters)
- Artist Onboarding Form
- (Optional) Manager Dashboard

## Getting Started

```bash
npm install
npm run dev
```
